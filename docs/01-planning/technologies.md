# Technologies Used

| Technology           | Justification                                                                                                                               |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| React                | Most popular frontend framework, large community, component-based architecture, virtual DOM for performance, easy to learn and use.          |
| TypeScript           | Adds static typing to JavaScript, improves code maintainability and scalability, catches errors early, integrates well with React.           |
| Material UI/Ant Design | Provides pre-built and customizable UI components, speeds up development, ensures consistent and modern design.                                |
| Redux Toolkit/Zustand | Simplifies state management in React applications, makes it easier to manage complex application states.                                    |
| Spring Boot          | Popular Java framework for building web applications and microservices, simplifies configuration, provides many features out of the box.    |
| FastAPI              | Modern Python framework for building APIs, high performance, easy to learn and use, automatic data validation and documentation.              |
| PostgreSQL           | Powerful and reliable open-source relational database, widely used, good performance, supports complex queries and transactions.                |
