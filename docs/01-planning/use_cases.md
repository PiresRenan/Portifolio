# Use Cases

| Use Case              | Actor               | Description                                                                                     |
| --------------------- | ------------------- | ----------------------------------------------------------------------------------------------- |
| Create an Account     | User                | The user enters their information (name, email, password) and creates a new account.           |
| Log In                | User                | The user enters their credentials (email, password) to access their account.                     |
| View Profile          | User                | The user views their profile information, including projects and contact details.               |
| Edit Profile          | User                | The user updates their profile information.                                                      |
| Add a Project         | User                | The user adds a new project to their portfolio, providing details and optionally a link.       |
| Edit a Project        | User                | The user modifies the details of an existing project in their portfolio.                        |
| Delete a Project      | User                | The user removes a project from their portfolio.                                                |
| Send a Message        | Visitor             | The visitor fills out the contact form and sends a message to the website owner.                 |
| View Messages (Admin) | Administrator (opt.) | The administrator views and manages messages sent through the contact form.                    |
| Manage Users (Admin)  | Administrator (opt.) | The administrator adds, edits, or deletes user accounts, and manages their permissions.         |
| Manage Projects (Admin) | Administrator (opt.) | The administrator reviews and approves/rejects projects submitted by users (if moderation is enabled). |
